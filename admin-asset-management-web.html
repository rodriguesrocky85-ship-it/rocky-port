THE PROCESS

Asset Management Web

Web-based admin platform for centralized asset governance, configuration workflows, and operational controls.

Project Overview

A web-based administrative dashboard built to help operations managers monitor asset data, usage cycles, and reporting insights across infrastructure projects. The dashboard provides a centralized platform where management can track asset distribution, monitor lifecycle usage, and review operational reports.

ROLE
UI/UX Designer
RESPONSIBILITIES
UX Research, Information Architecture, Dashboard Design, UI Design
TOOLS
Figma Photoshop
PLATFORM
Web Dashboard

Problem

Management lacked a centralized system to monitor asset data across multiple locations.

Key challenges

  • No real-time visibility into asset distribution
  • Difficulty tracking asset usage cycles
  • Manual reporting processes
  • Lack of operational insights for decision making
  • Limited visibility into asset performance and usage

Research

Understanding how managers monitored operations helped define the dashboard structure.

Key insights

  • Managers required clear reporting and summarized insights
  • Asset usage cycles needed visual representation
  • Data needed to be presented in a simple and structured format

Direction

  • Create a dashboard focused on operational visibility
  • Emphasize reporting clarity and lifecycle tracking

Solution

I designed a web-based dashboard that provides management with clear insights into asset operations.

Key features

  • Asset overview dashboard with operational summaries
  • Usage cycle tracking for infrastructure assets
  • Reporting modules for operational insights
  • Structured data visualization for easy monitoring

Screens

Outcome

The dashboard enabled management teams to monitor asset usage and operational reports from a centralized system.

Key improvements

  • Improved operational visibility
  • Faster reporting and data access
  • Better asset lifecycle monitoring
